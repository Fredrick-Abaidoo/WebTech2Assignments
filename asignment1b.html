Exercise 3: Number Operations

3a: Collect and Display a Number

Collect a number using prompt() and display it using alert().

// JavaScript code
var number = prompt("Please enter a number:");
alert("You entered: " + number);

3b: Multiply Two Numbers

Collect two numbers and display their product.

// JavaScript code
var num1 = parseFloat(prompt("Enter first number:"));
var num2 = parseFloat(prompt("Enter second number:"));
var product = num1 * num2;
alert("The product of " + num1 + " and " + num2 + " is " + product);

3c: Find the Larger Number

Collect two numbers and display the larger one.

// JavaScript code
var num1 = parseFloat(prompt("Enter first number:"));
var num2 = parseFloat(prompt("Enter second number:"));
var larger = (num1 > num2) ? num1 : num2;
alert("The larger number between " + num1 + " and " + num2 + " is " + larger);
